Games

Here are a selection of games either finished or finished enough to warrant talking about.
These include games from game jams, games as projects for university, and my own personal game projects.

  • Platforms: Windows, WebGL
  • Release Date: January 2021

Scavenger Showdown was created in 48 hours as part of the 2021 Global Game Jam.
The theme for the game jam was "Lost and Found"; our group created a competitive collectathon party game where each player must find a set of items in a specific order before everyone else. Combat enhances this gameplay by allowing players to attack anyone nearby, making them lose precious time.
The game can also be played single-player, where it becomes a race against the clock for the best speedrun time.

For Scavenger Showdown, I worked on:

  • game design/balance
  • gameplay programming
  • UI programming
  • music (listen to the OST here!)

Canvas

  • Platforms: Windows, MacOS
  • Release Date: 2021 (demo)

Canvas is a party rhythm game where the aim is to coat the most surfaces as possible with your team's paint.

Canvas has been presented twice as part of the AVCon Indie Games Room, in 2017 and 2019.

For Canvas, I worked on:

  • game design/balance
  • gameplay programming
  • UI design / programming
  • art (2D/3D/UI)
  • music (listen to some of the OST here!)

  • Platforms: Windows (SteamVR-compatible hardware)
  • Release Date: 2020

Built-Up! is a VR building game about creative construction and coffee-cup relocation.
Using simple building blocks and connectors, players create innovative solutions to various problems to knock over coffee cups in various locations.
I assisted in creating this game as lead programmer and composer alongside six other developers as part of my Virtual Reality Design and Development II course at The University of Adelaide.

For Built-Up! , I worked on:

  • game design/concept
  • gameplay programming
  • physics programming
  • SteamVR integration
  • music (listen to the OST here!)

RAMiGo

  • Platforms: Windows (VRTK-compatible hardware)
  • Release Date: 2020

RAMiGo (short for Rooftop Apocalyptic MiniGolf), is a VR mini-golfing experience.
Players must navigate golf balls through 9 increasingly-difficult mini courses reminiscent of classic mini-golf.
I created this game independently as part of my Virtual Reality Design and Development I course at The University of Adelaide.

For RAMiGo, I worked on:

  • game design/concept
  • gameplay programming
  • physics programming
  • 3D modelling/rigging
  • VRTK integration
  • music (listen to the OST here!)

Downtown Patrol Force

  • Platforms: Windows, MacOS
  • Release Date: Unreleased (developed mostly in 2019)
It's 1982. The Downtown Patrol Force are looking for new members. Will you join them on this all new crazy adventure?

Downtown Patrol Force is a top-down 3D open-world action-adventure game about "cop show" tropes/cliches. With up to 4-player local co-op, players would patrol Mahogany Bay, Miamifornia and assist citizens while fighting crime.

For Downtown Patrol Force, I worked on:

  • game design/concept
  • environment art
  • physics objects
  • Textalogue, the custom dialogue engine for Unity 3D
  • objective system + gameplay programming
